# View

Frontend React application (user interface).

**Tech Stack:**
- React + TypeScript
- Vite (build tool)
- Tailwind CSS + shadcn/ui

**Main Components:**
- `src/pages/` - Page components (Index, Results, Login, etc.)
- `src/components/` - Reusable UI components (CarCard, SearchChat, etc.)
- `src/services/api.ts` - API client for backend communication

**What it does:**
- Displays the car search interface
- Sends search queries to the backend API
- Shows car results in cards
